> This NPM Package help you to generate starter files for express application. It installs useful packages out of the box. You now need to focus on development rather than spend time setting up the project.

<h2>üßê Features</h2>

Here're some of the project's best features:

*   Creates Basic Folder structure of the express application
*   Created Public folder and CSS JS Images subdirectories
*   Installs nodemon express ejs mongoose cookie-parser

## Setup

<p>1. Install the tool</p>

```sh
npm i my-express-generator
```

<p>2. Generate the Project</p>

```sh
npx create-express-app (appName)
```
